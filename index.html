📱 隐私政策

速成英语 (Quick English)

最后更新：2026年4月18日

📋 概述

速成英语是一款完全本地运行的英语学习应用。我们重视您的隐私，本应用不会收集、上传或分享您的个人信息。

核心原则： 您的数据属于您自己，我们不会收集、存储或分享您的任何个人信息。

🔒 数据收集

本应用不收集任何个人身份信息，包括但不限于：

  • 姓名、邮箱、电话号码等个人信息
  • 设备标识符（IDFA、IDFV等）
  • 位置信息
  • 使用行为数据
  • 录音文件

💾 本地存储

应用会在您的设备上本地存储以下信息，这些数据仅存储在您的设备上，不会上传到任何服务器：

  • 应用设置：主题模式（深色/浅色）、语音偏好、语速等
  • 学习进度：您的学习记录和进度
  • 内购状态：是否已购买支持作者功能的本地标记
  • 更新提醒：您选择"稍后提醒"的时间戳

这些数据随应用卸载而自动删除。

🎤 权限使用

麦克风权限

用途：录制您的跟读发音，用于即时播放对比，帮助您改善发音。

数据处理：

  • 录音文件仅临时存储在设备的临时目录中
  • 录音文件不会上传到任何服务器
  • 录音文件在应用关闭或重新录制时自动删除
  • 您可以随时在系统设置中撤销麦克风权限

🌐 网络使用

应用仅在以下情况访问网络，且不会上传您的任何个人数据：

1. 检查应用更新

应用会访问 Apple App Store 的公开 API 检查是否有新版本可用。此请求不包含任何个人信息。

2. 内购交易

当您选择支持作者时，支付流程完全由 Apple 的支付系统处理。我们不会获取或存储您的支付信息。

💳 内购说明

应用内购买（支持作者功能）通过 Apple 的 App Store 支付系统处理：

  • 所有支付信息由 Apple 处理和存储
  • 我们不会获取您的信用卡、支付宝或其他支付信息
  • 购买记录仅在您的设备上本地标记
  • 购买是一次性的消耗型项目，用于支持开发者

🚫 第三方服务

本应用不使用任何第三方服务，包括：

  • 数据分析工具（如 Google Analytics、Firebase Analytics）
  • 广告平台
  • 用户行为追踪工具
  • 社交媒体 SDK
  • 崩溃报告工具

应用仅使用 Flutter 官方插件和 Apple 系统 API。

👶 儿童隐私

本应用适合所有年龄段使用，不会收集儿童的个人信息。由于应用不收集任何个人数据，因此符合儿童在线隐私保护法（COPPA）的要求。

🔐 数据安全

由于应用不收集或上传任何个人数据，因此不存在数据泄露的风险。所有数据都安全地存储在您的设备上，受 iOS 系统的安全机制保护。

✅ 您的权利

您对自己的数据拥有完全控制权：

  • 访问权：所有数据都存储在您的设备上，您可以随时访问
  • 删除权：卸载应用即可删除所有本地数据
  • 撤销权限：您可以随时在系统设置中撤销麦克风权限
